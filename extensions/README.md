# Расширения (Extensions)

Эта директория содержит расширения, созданные агентом.

## tools/
Кастомные инструменты. Каждый `.py` файл должен содержать функцию:
```python
def register() -> list:
    return [MyTool()]
```

## adapters/
Кастомные LLM-адаптеры. Аналогичная структура с `register()`.

## scripts/
Вспомогательные скрипты, вызываемые агентом через shell.

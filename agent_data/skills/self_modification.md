# Навык: Самомодификация

## Описание
Навык позволяет агенту модифицировать собственный код, создавать расширения,
обновлять knowledge-файлы и перезапускаться для применения core-изменений.

## Когда применять
- Пользователь просит научиться чему-то новому
- Нужно добавить новый инструмент или адаптер
- Нужно изменить правила, навыки или память
- Нужно исправить баг в собственном коде

## Шаги

### Добавление навыка (MD)
1. Найти документацию (web_search + web_fetch)
2. Сформировать описание навыка
3. Записать в agent_data/skills/{name}.md через self_modify(action=update_knowledge)

### Добавление инструмента (PY)
1. Найти документацию и примеры
2. Написать Python-код с классом, наследующим BaseTool
3. Записать в extensions/tools/{name}.py через self_modify(action=create_extension)
4. Перезагрузить реестр tools

### Изменение core-кода
1. Прочитать текущий код через self_modify(action=read_source)
2. Внести изменения через self_modify(action=write_source)
3. Изменения автоматически коммитятся в git
4. Вызвать self_modify(action=restart) для перезапуска

## Необходимые инструменты
- self_modify
- file_ops
- web_search (для поиска документации)
- web_fetch (для чтения документации)
- shell (для установки зависимостей)

# Evo-Agent

Легковесный самомодифицирующийся AI-агент на Python.

## Возможности

- **Telegram-интерфейс** -- общение через бота
- **LLM с function calling** -- OpenAI-совместимый протокол (OpenAI, Ollama, vLLM, OpenRouter, LM Studio)
- **Инструменты** -- shell, файловые операции, веб-запросы, самомодификация
- **Knowledge система** -- agent.md, rules.md, memory.md, skills/
- **Автономность** -- 4 уровня, от полного подтверждения до полной автономии
- **Самомодификация** -- агент может менять свой код, навыки и расширения

## Быстрый старт

### 1. Установка

```bash
pip install -e .
```

### 2. Конфигурация

Скопировать `.env.example` в `.env` и заполнить:

```bash
cp .env.example .env
```

Настроить `config.yaml` -- LLM endpoint, модель, Telegram токен.

### 3. Запуск

```bash
python -m evo_agent
```

## Структура

```
evo-agent/
├── src/evo_agent/          # Ядро агента
├── agent_data/             # "ДНК" -- знания, правила, навыки
├── extensions/             # Расширения (создаются агентом)
├── workspace/              # Рабочая директория
├── data/                   # Runtime-данные
├── config.yaml             # Конфигурация
└── .env                    # Секреты
```

## Команды Telegram

- `/start` -- приветствие
- `/autonomy <0-3>` -- уровень автономности
- `/status` -- текущий статус
- `/skills` -- список навыков
- `/memory` -- просмотр памяти

## Уровни автономности

| Уровень | Название | Поведение |
|---------|----------|-----------|
| 0 | Paranoid | Подтверждение на каждое действие |
| 1 | Careful | Подтверждение на опасные действия |
| 2 | Balanced | Подтверждение только на самомодификацию |
| 3 | Autonomous | Полная автономия |
